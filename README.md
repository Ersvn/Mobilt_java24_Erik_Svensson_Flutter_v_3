## Flutter Mood

Flutter Mood är en enkel crossplattform-app byggd i Flutter för kursen Mobilutveckling med Java.

Appen låter användaren skriva en kort text om hur dagen känns, välja ett humör och spara detta i Firebase Cloud Firestore. Den sparade datan visas sedan på en separat sida i appen.

### Krav/Funktioner

- Fungerar på Android och webb
- Två sidor med navigation
- Textfält för användarens input
- Dropdown för att välja humör
- Knapp för att spara data
- Sida som visar sparade humör
- Bild via webblänk
- Firebase Cloud Firestore som databas
- Custom app icon och favicon

### Tekniker

- Flutter
- Dart
- Firebase
- Cloud Firestore

---

### Plattformar

Appen är testad på:

- Android-emulator
- Webbläsare via Flutter web

### Bygga projektet

För att köra projektet:

1. flutter pub get 
2. flutter run

#### För att bygga APK:

flutter build apk

#### För att bygga webbversion:

flutter build web
